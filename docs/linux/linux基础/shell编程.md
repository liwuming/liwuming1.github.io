总结正则的书写思路

是否需要锚定开始和结束
行锚定
词锚定

例如：找出/etc/passwd文件中root或者centos或者xx相关用户信息
```bash
grep -e "^(root|)"

```
首先，因为需要用到或，所以需要添加-e选项，扩展正则表达式
/etc/passwd文件中一行表示一个用户记录，为了防止匹配到root组而非root账户的记录行，因而需要行首锚定--^
root|centos|xx是一个完整的账户，因而需要词尾锚定--\>
