关于shell脚本的执行方式

# 输入i,进入编辑模式
echo 'hello world';
# 输入:wq，保存退出

赋予脚本执行权限，将其变为可执行文件

chmod -x bash.sh
./bash.sh

注意：点号--.必不可少，少了点号，解释器就会把bash.sh当成命令，在系统常量--PATH下进行查找，若当前目录不在path路径下，则会报如下错误提示：-bash: bash.sh: command not found

作为shell命令的参数

bash bash.sh

此时脚本可不具备执行权限，但首行应为shebeng--#!开头，执行注明脚本的解释器类型
