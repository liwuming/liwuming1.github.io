配置文件的作用

不同平台下的配置文件所在位置有所差异

在windows平台下配置文件为my.ini，在linux平台下的配置文件为my.cnf 在linux下的配置一般放在/etc目录下，因而mysql的配置文件my.conf一般放在/etc/my.cnf,

/etc/my.cnf /etc/mysql/my.cnf /usr/local/mysql/etc/my.cnf ~/.my.cnf
