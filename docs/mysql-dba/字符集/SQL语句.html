use，告诉mysql使用指定的数据库作为后续语句的默认数据库

语法： use db_name;

在登录mysql之后，就好比使用bash的cd命令切换到了数据库的数据根目录， 数据库是在数据目录下的一个目录，使用use命令就好比使用cd命令切换到db_name文件夹下，这需要指定的权限

在后续语句中是否可以非db_name中的表，当然阔以，只是使用db_name库下的表无需添加库前缀，而使用非db_name库的表需要加前缀other_db_name.table即可
