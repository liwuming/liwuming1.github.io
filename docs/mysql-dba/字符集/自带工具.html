如果在数据库初始化时使用的是mysqld命令的--initialize-insecure选项可无需命令登录，若使用的是--initialize选项需要输入临时密码登录，且第一件事是修改密码才能进行操作

如何修改密码呢?
- 1、set password [for user@host] = password('<password>');

如果是为自己设置密码，则for user@host可省略

- 2.alter

- 3、update

- 4、mysqladmin -u -p password
