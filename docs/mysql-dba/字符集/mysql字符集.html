什么是字符集

字符集，顾名思义，就是字符的集合呗，那字符是什么鬼？

mysql支持很多字符集，常见的gbk,utf8,utf8mb4,其中mysql默认的字符集latin1，所以在一定要改过来 gbk是国家字符集，仅支持中文和英文，所有字符占用两个字节， utf8是国际字符集，支持所有全世界的所有字符，更加通用，因此使用广泛，英文字符占用单字节，其他字符占用3字节 utf8mb8是utf8的升级版，添加了对表情包的支持，非英文字符占用单字节，其他字符占用4字节，除非业务有表情包需求，否则还是建议utf8

mysql的字符集有多种级别

服务器字符集，

字符集配置

[client] default-character-set=utf8

[mysqld] character-set-server=utf8;
